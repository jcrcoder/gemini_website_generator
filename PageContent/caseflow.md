We developed CaseFlow as an out-of-the-box application for automating data lifecycle management in Relativity. It continuously monitors user activity to determine the proper workspace or storage for case data and then transitions that data appropriately. Save time and reduce costs associated with the wrong data sitting in the wrong stage of your case.

## Features

_Show an image or screenshot next to each of the three features_

### Automatically Move Workspaces

When you're spending most of your time loading documents for review, it's easy to forget about existing case data. CaseFlow can be set up to automatically move workspaces into a repository workspace, cold storage, or staging based on how that data's being used. You can also schedule upcoming transitions from a single interface.

### Robust Notifications

CaseFlow will proactively notify stakeholders before cases are moved, and users can pause automated actions directly from their email notifications if you need more time with your case. You can also view all upcoming events and historical actions via a single reporting dashboard.
